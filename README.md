# MarumaVR180

UnityおよびVRChat向けのVR180立体視映像表示システムです。

## 概要

独自の4分割アトラスマッピング技術を用いて、高解像度かつ低歪みなVR180映像を表示します。VRChatのカメラモード判定に対応し、ミラーやデスクトップ画面での不要な映り込みを制御します。

## 主な機能

- **VR180立体視**: 左右の視差に対応したリアルな立体映像表現。
- **独自プロジェクション**: 極座標変換と4分割アトラスマッピングによる効率的なUV展開。
- **視界制御**:
  - `_VRChatCameraMode` による制御（VRモード、デスクトップモード、ミラー等での表示切り替え）。
  - アスペクト比に基づく自動マスキング。

## ディレクトリ構成

プロジェクトの主要ファイルは `Assets/Marumasa/MarumaVR180` 配下に格納されています。

- `Shaders/`: 映像投影・制御用シェーダー（`VR180-Camera.shader` 等）
- `RenderTextures/`: 映像ソース用RenderTexture
- `Prefabs/`: セットアップ済みプレハブ
- `Animation/`: 制御用アニメーションクリップ

## 使い方

1. `Prefabs` フォルダ内のプレハブをシーンに配置します。
2. カメラまたは映像ソースを `RenderTexture` に割り当てます。
3. マテリアル設定で投影パラメータを調整します。
